# react-native-bridtv-sdk

react native module for bridtvSDK

## Installation

```sh
npm install react-native-bridtv-sdk
```

## Usage

```js
import { BridtvSdkView } from "react-native-bridtv-sdk";

// ...

<BridtvSdkView color="tomato" />
```
